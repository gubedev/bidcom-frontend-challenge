## Usos frecuentes

| Elemento | Clases |
|---|---|
| Marca / categoría | `text-sm uppercase tracking-widest font-medium text-gray-500` |
| Título de producto | `text-2xl sm:text-3xl font-bold text-body-text leading-tight` |
| Precio | `text-3xl font-bold text-brand` |
| Descripción | `text-sm sm:text-base text-gray-600 leading-relaxed` |
| Label de tabla | `text-sm text-gray-500` |
| Footer heading | `text-sm font-semibold uppercase tracking-wide text-white` |
